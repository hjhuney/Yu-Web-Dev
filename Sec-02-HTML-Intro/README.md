# Section 2 + 3: HTML

## Heading Tags

```
<h1> through <h7>
```

## Horizontal Line

Create with

```
<hr>
```

Increase size of line:

```
<hr size = "3">
```

Size is an HTML attribute. 

## Comments

```
<!-- comments go here
Also here
blah -->
```

## Encoding

UTF-8 is generally the standard; includes all characters. 

```
<meta charset="UTF-8">
```

## HTML Meta Tags

Description of web site:

```
<meta name="description" content="description of web page"
```
